Навыки

HTML

Первые знания HTML я получил еще во время обучения в университете, используя их при создании онлайн-опросников, форм тестов и информационных страниц. Позднее я также применял их при создании сайтов с помощью визуальных редакторов и инструментов. В настоящее время в процессе верстки я работаю непосредствеено с HTML кодом

CSS

Системное представлении о применении CSS и адаптивной верстки я приобрел, пройдя несколько обучающих курсов в 2019/ 2020 году и получив практический опыт верстки сайтов. В настоящее время я применяю синтаксис SCSS и осваиваю предоставляемые им возможности. Использую технологии CSS Flex и CSS Grid, изучаю возможности анимации и SVG.

JS

В настоящее время я нахожусь в процессе активного изучения, имею опыт применения Javascript и JQuery для работы с элементами DOM, например для скрытия и отображения отдельных элементов и блоков на веб-странице, настройки интерфейса и проверки данных при заполнении пользовательских форм и переключения оформления.

Свои первые сайты я разрабатывал на основе конструкторов, таких как Platforma LP и Tilda. Примеры моих работ можно увидеть на Behance и Github

Этот сайт-порфолио я разработал уже самостоятельно. Для повышения производительности я использовал Gulp. Для отдельных элементов на страницы подключены библитеки JQuery и SlickSlider. При оформлении страницы в работе со стилями использую SCSS, применяю иконочные шрифтами и SVG-изображениям.

Имею опыт работы в операционных системах Windows и LinuxMint, использую основные команды терминале при установке пакетов, при работе с NPM и Git.

Имею опыт работы в графических редакторах Adobe Photoshop, GIMP и Inkscape. При работе с макетами также применял Adobe Fireworks и Figma.
